libbbs

Specification-compliant and performant implementation of the BBS signature scheme.

Provides a library libbbs implementing the BLS12381-SHA-256 and BLS12-381-SHAKE-256 cipher suite.

Prerequisites

Dependencies:

  • cmake (build only)
  • gmp (optional, faster than the default)
mkdir build
cd build
cmake .. # Without GMP or
cmake -DRELIC_ARITH="gmp" .. # (if you have gmp)

Installation

Within build:

make install

Test

Within build:

make
make test

Benchmark

Within build:

./test/bbs-test-bench
